# 🧠 Cofre de Neurônios — Guia Rápido

## 🚀 Como começar

1. Abra o Obsidian → clique em **"Open folder as vault"**
2. Selecione a pasta `CELEBRO DO BRUNO`
3. Instale os plugins recomendados (veja abaixo)
4. Abra `[[HOME]]` como sua página inicial

---

## 📁 Estrutura

| Pasta | Para que serve |
|-------|---------------|
| `00_INBOX` | Captura rápida — jogue qualquer coisa aqui |
| `01_MAPAS` | Índices (MOC) de cada área de conhecimento |
| `02_IDEIAS` | Notas em processamento, organizadas por tema |
| `03_NOTAS_LITERATURA` | Resumos de livros, artigos, cursos, vídeos |
| `04_NOTAS_PERMANENTES` | Seu conhecimento sintetizado, em suas palavras |
| `05_TEMPLATES` | Modelos prontos para criar notas |
| `06_RECURSOS` | Imagens e anexos |

---

## 🔌 Plugins Recomendados

Vá em **Configurações → Community Plugins → Browse** e instale:

1. **Dataview** — consultas dinâmicas nas notas (tipo SQL para notas)
2. **Templater** — templates mais poderosos que o padrão
3. **Calendar** — visualização de daily notes em calendário
4. **Tag Wrangler** — gerencia e renomeia tags em massa
5. **Obsidian Git** — backup automático no GitHub
6. **Kanban** — quadro kanban para gerenciar projetos

---

## ⌨️ Atalhos Configurados

| Atalho | Ação |
|--------|------|
| `Cmd/Ctrl + D` | Criar/abrir daily note de hoje |
| `Cmd/Ctrl + B` | Negrito |
| `Cmd/Ctrl + I` | Itálico |
| `Alt + ←/→` | Navegar histórico |

---

## 🧭 Fluxo de Trabalho (Método Zettelkasten)

```
1. CAPTURAR  → 00_INBOX (sem pensar, rápido)
2. PROCESSAR → 02_IDEIAS (dar contexto)
3. REFERENCIAR → 03_NOTAS_LITERATURA (de onde veio)
4. SINTETIZAR → 04_NOTAS_PERMANENTES (seu pensamento próprio)
```

**Regra de ouro:** Cada nota deve ter pelo menos **2 links** para outras notas.
Isso cria a teia que transforma anotações em conhecimento.

---

## 💡 Dicas

- Use `[[double brackets]]` para linkar qualquer nota
- Use `#tags` para categorizar
- Use `> ` para citações/insights importantes
- Use `- [ ]` para tarefas
- Abra o **Graph View** (`Cmd+G`) para ver sua teia de conhecimento crescer
